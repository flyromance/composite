dom事件

dom事件的级别
  • 0级 dom.onclick
  • 1级 dom1标准设计的时候，没有设计与事件相关的东西，所以直接是dom2级事件
  • 2级 dom.addEventListener('click', function() {}, false), dom.attachEvent('onclick', function () {})
  • 3级 添加了很多事件类型，如mouseup、keyup等等

dom事件流
浏览器设计的一套机制：用户和网页的交互方式，比如说点击，顶层先捕获，目标阶段，冒泡到顶层
dom捕获的具体流程

  • 第二个是document
  • document.documentElement
  • document.body
  • ...

  • event.target event.srcElement
